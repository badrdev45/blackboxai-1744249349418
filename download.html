Download Game (ZIP)

Installation Instructions:

  1. Download FPSGame.zip
  2. Extract all files
  3. Open FPSGame.sln in Visual Studio 2019 or later
  4. Build the solution (F7)
  5. Run the game (F5)

Requirements:

  • Windows 10 or later
  • Visual Studio 2019 or later
  • DirectX 11 compatible graphics card

Controls:

  • WASD: Movement
  • Mouse: Look around
  • Space: Jump
  • ESC: Exit/Pause
  • Left Mouse Button: Shoot
